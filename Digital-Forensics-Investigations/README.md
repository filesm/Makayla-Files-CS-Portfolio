# Digital Forensics Investigations

Conducted digital forensic investigations using Autopsy and Wireshark to analyze network traffic, deleted files, and system artifacts.

## Overview
This project involved investigating digital evidence from forensic lab scenarios, including deleted files, system activity, and network traffic.

## Skills Used
- Cybersecurity
- Digital Forensics
- Autopsy
- Wireshark
- Network Traffic Analysis
- Evidence Documentation
